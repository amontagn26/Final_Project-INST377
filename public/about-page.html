What this app does:

This application is an analysis tool to search and track your favorite TV shows. The app has two key functions both located on the Project Page. The first function allows users to search any TV show to learn information on the show and chart the average IMBD rating of each of the show's episodes. The second function allows users to favorite the searched show to add to a favorite shows table.

Data Sources

  • TV Maze - API that provides live TV show data
  • Supabase - Cloud database to store and recieve favorite shows

How to Use:

  1. Navigate to Project Page
  2. Input show name in search bar
  3. Click favorites button

Development Background:

This app was created to provide users with a simple way to search for TV shows, explore episode ratings and save their favorite shows for others to see.
